# glyph.kt

The Kotlin rewrite of the Glyph Discord bot.

Glyph is a Discord bot with a powerful starboard system, link embed generation, and various other fun features.

To learn more about how to use Glyph, check out the documentation [here](https://glyph.yttr.org/).

## Self-Hosting

Hosting your own copy of Glyph requires some setup.

1. Create a MariaDB database and Redis data store
2. Set the necessary environment variables as seen in the application.conf files
3. Build and start the bot and website
